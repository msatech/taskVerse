# **App Name**: TaskVerse

## Core Features:

- Authentication and User Management: Secure sign-up, login, and password reset flows with email/password authentication. Includes user roles (USER, ADMIN) and organization roles (OWNER, ADMIN, MEMBER).
- Project and Issue Management: Create, manage, and track projects, issues (stories, tasks, bugs, epics), and sprints. Includes support for issue types, statuses, priorities, assignees, and due dates.
- Backlog and Board Views: Kanban and Scrum board views with drag-and-drop functionality for managing issues. Backlog page for unsprinted issues with inline creation and ordering.
- Issue Detail View: Detailed view for each issue with inline editing of title, description, priority, status, and assignee. Includes support for comments, attachments, and activity log.
- Sprint Management: Create, edit, start, and complete sprints for Scrum projects. Validate sprint start with at least one issue and set start and end dates.
- Reports and Analytics: Basic reports including burndown chart for active sprint, issues by status (bar chart or donut), and throughput (issues completed per sprint).
- Intelligent Issue Tagging: Use a generative AI tool to automatically suggest tags for new issues based on their description.

## Style Guidelines:

- Primary color: Dark teal (#008080) for a professional and calming feel, fitting for project management.
- Background color: Dark gray (#222222) to support dark mode and reduce eye strain.
- Accent color: Soft orange (#FFA500) to draw attention to important actions and highlights.
- Body font: 'Inter', a grotesque-style sans-serif with a modern, machined, objective, neutral look; suitable for body text.
- Headline font: 'Space Grotesk', A proportional sans-serif with a computerized, techy, scientific feel, for headlines.
- Code font: 'Source Code Pro' for displaying code snippets.
- Use clear, consistent icons from a library like Font Awesome or Material Icons for issue types, priorities, and actions.
- Jira-inspired layout with left sidebar navigation and top bar for search and user settings. Responsive design for mobile and desktop.
- Subtle animations for transitions and interactions, such as drag-and-drop and modal appearances.
