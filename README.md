# TaskVerse

This is a Jira-style project management & issue tracking platform built with Next.js and shadcn/ui.

## Getting Started

First, install the dependencies:

```bash
npm install
```

Next, run the development server:

```bash
npm run dev
```

Open [http://localhost:9002](http://localhost:9002) with your browser to see the result.

You can start by exploring the landing page at the root URL or navigate to `/dashboard` to see the application shell and the intelligent issue tagging feature.
